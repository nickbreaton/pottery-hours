# Pottery Hours

A bespoke app which converts a pottery schedule distributed as a Google Sheet into a subscribable calendar feed.

<br>
<p align="center">
  <i>Dedicated to and built for my wonderful wife, Sydney,<br>who makes lovely and inspiring <a href="https://kilnmeplease.com/">pottery</a>.</i>
</p>
<br>

## Getting started

Install dependencies:

```bash
npm install
```

Run the development build:

```bash
npm run dev
```
